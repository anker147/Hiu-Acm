## HIU-ACM 全栈审查与 BUG 修复 + 性能优化

用户要求"跑一下整个项目，优化所有可优化的地方，修复所有 BUG"。对 HIU-ACM 集训管理系统（Cloudflare Workers + D1 + GitHub Pages SPA）做了全栈审查并修复。

### 修复的 BUG（按严重度）
1. **字段名不匹配（致命）**：后端 `problem_stats` 返回 `selected_count`/`completed_count`，前端访问 `selectedCount`/`completedCount`，导致所有题被判为"新题"、统计显示全 0。在 `app.js loadData` 做字段映射修复。
2. **重复方法定义**：`app.js renderTodayTasksOrSelection` 在 107 行和 497 行重复定义，后者覆盖前者导致 `selectionPhase` 不重置。删除重复。
3. **搜索框失焦**：`oninput` 触发整页 re-render 导致 input 失焦无法连续输入。新增 `renderProblemGridOnly` 只刷新题目网格。
4. **选题超限误删**：`toggleProblem` 超过 20 题时 `shift()` 删了最早选的，改 `pop()` 删刚选的。
5. **history 除零**：题单题目被删空后 `pct=NaN`，加守卫。
6. **admin showUserTasks name 丢失**：重调时传空 name 致标题空，改为从 `_usersCache` 查。
7. **editGroup JSON 嵌入 HTML 脆弱转义**：改为按 id 从 `_groupsCache` 查数据，彻底消除注入风险。
8. **后端登录日志重复调用 `handleLogin`**：try/catch 异常时二次调用，登录逻辑跑两次；第 804 行还有死代码。重写为只调一次。
9. **后端 `/api/me` 每次刷新记登录日志**：污染日志，移除。
10. **api.js 响应解析无容错 + 无超时**：加 try/catch + AbortController 15s 超时。
11. **auth.js JWT base64 缺 padding**：`atob` 可能失败，补齐 padding。

### 性能优化
- **IP 归属地查询灾难**：admin 日志页对每条记录实时调 `ip-api.com`（100 条 = 100 次外部请求 + 3s 超时，触发限流）。改为登录时计算 `region`/`device` 入库，查询时直接读，历史空数据 `enrichLog` 按需回写。新增 `migrations/002_login_logs_region.sql`。
- **dashboard N+1 查询**：`handleAdminDashboard` 逐用户/小组查 `daily_tasks`。改为 5 个并行查询 + 内存聚合。
- **粒子 canvas 高 DPI 模糊**：加 `devicePixelRatio`（限 2 倍）+ `setTransform`。
- **触屏设备自定义光标**：`pointer:coarse` 跳过光标并恢复默认，避免 JS 失败时桌面光标消失。

## 第二轮：删除题目BUG修复 + 题库重置功能 + UI优化

用户反馈三个问题：1.优化按钮与UI 2.题库重置为0+清除题集选择 3.管理员删除题目仍有问题。

### 修复3：管理员删除题目BUG（根因：problemId 类型不一致）
- **根因**：`daily_tasks.problems` 存的是字符串数组 `["1001"]`，但后端 DELETE 路由 `parseInt(problemsMatch[3])` 转成数字 `1001`，导致 `"1001" !== 1001`，`filter` 和 `includes` 全部失效，题目删不掉。
- **修复**：后端统一用 `String()` 转换比较。涉及 4 处：
  - `handleAdminUsers` DELETE problems 路由：`problemId` 不再 parseInt，problems/completed 数组 `.map(String)`
  - `handleCompleteProblem`：统一 `pid = String(problemId)`
  - `complete-all` 批量完成：改为单次查询+内存处理（顺便优化 N+1），统一 String 比较
  - PUT 路由的完成状态修改：统一 String 比较
- 前端 `admin.js`：`deleteTaskProblem` onclick 的 `${id}` 加引号防数字传入；`batchCompleteAll` 去掉 `parseInt` 传字符串。

### 新增：题库重置功能
